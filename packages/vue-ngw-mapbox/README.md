# Vue ngw map component for mapbox

Plugin to integrate NGW maps into vue.js with MapboxGL-js adapter

```html
<vue-ngw-map
  fullFilling
  baseUrl="https://demo.nextgis.com"
  webmapId="4251"
  qmsId="487"
></vue-ngw-map>
```
