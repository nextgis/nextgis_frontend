# Paint

An auxiliary library for working with the style of vector layers

## Installation

```bash
# latest stable
$ npm install --save-dev @nextgis/paint
# or
$ yarn add @nextgis/paint
```

## Usage

```js
import { isPaint } from '@nextgis/paint';
```
