# Dom

Collection of libraries for working with the DOM.

## Installation

```bash
# latest stable
$ npm install --save-dev @nextgis/dom
# or
$ yarn add @nextgis/dom
```

## Usage

```js
import * as DOM from '@nextgis/dom';

```
